# P.I.G. Naming Conventions

## Quote Image File Names

Use this structure for Button Poetry quote image exports:

```text
AUTHOR LAST NAME - BOOK SHORTENER - QI - POEM TITLE - # OF QI
```

Example:

```text
WAKEFIELD - SW - QI - NEXT LIFE SOUNDTRACK - 1
```

## Book Shortener Rules

1. If a book title has more than two words, use up to four initials, including articles.
2. If a book title has exactly two words, use both initials, including articles.
3. If a book title has one word, use the first four letters of that word.
4. Exclude the word `and` and the character `&`.
5. Use known catalog overrides when P.I.G. or Button Poetry history has a canonical shortener that differs from the rule output.

## Known Overrides

```text
A Choir of Honest Killers -> ACHK
Don't Be Afraid to Be Bad -> DBAT
I'll Fly Away -> IFA
```

## Notes

- Author should be the author's last name in all caps.
- Book shortener should be all caps.
- Poem title should be all caps unless a specific catalog/title convention requires otherwise.
- `# OF QI` should indicate the quote image number/version for that poem or excerpt when multiple QIs exist.
